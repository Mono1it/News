# News Feed App (SwiftUI)

Pet-проект, созданный с целью углублённого изучения **SwiftUI** и работы со сторонним REST API.  
Проект разрабатывался как практическое дополнение к основному обучению, так как SwiftUI в рамках курса рассматривался поверхностно, несмотря на его активное использование в современных iOS-приложениях.

---

## Описание

Приложение представляет собой ленту новостей, данные для которой получаются из **News API**.  
Пользователю отображаются заголовки и темы новостей, а также изображения, загружаемые из сети.  
При выборе новости открывается экран с подробной информацией и возможностью перейти к полному материалу в источнике через встроенный **WebView**.

Полный текст новостей не парсится, так как он защищён авторским правом и содержит ограничения на стороне источников.

---

## Используемые технологии

- **Swift**
- **SwiftUI**
- **MVVM**
- **URLSession**
- **REST API / JSON**
- **Kingfisher** (загрузка и кэширование изображений)
- **WebView**
- **Swift Package Manager**

---

## Основной функционал

- Загрузка и отображение ленты новостей
- Группировка новостей по темам
- Асинхронная загрузка изображений
- Экран детального просмотра новости
- Открытие оригинального источника во встроенном WebView
- Обработка состояний загрузки и ошибок сети

---

## Цели проекта

- Углублённо изучить **SwiftUI**
- Закрепить работу с внешним REST API
- Попрактиковаться в архитектуре **MVVM**
- Понять особенности асинхронной загрузки данных и изображений
- Освоить работу со сторонними библиотеками через SPM


## Примечание

Проект создан в учебных целях и используется как pet-проект для демонстрации навыков работы со SwiftUI и сетевыми запросами.
